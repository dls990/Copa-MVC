Gerenciamento sistemático da copa

O sistema foi feito para analisar os dados dos jogos que estão ocorrendo, podem ser criados jogos quando quiser, colocando o placar, os times, seleções, e etc, utilizando um método de classificação automática, assim que criar um grupo, ele já aparece em outras paginas, na pagina resultados, voce pode registrar os pontos, fazendo que na pagina de classificação, diga qual time ganhou, quantos pontos foram marcados, e se houve empate ou não.

O projeto foi separado 8 pastas, sendo duas delas fora do sistema principal, as duas são uma apenas para mostrar o site em funcionamento, a outra é o SQL do Database e os outros 6 são o código principal do site, todas dentro da pasta "app".

O nosso banco de dados também recebe imediatamente os dados das partes que você coloca os times jogadores, contando gols, placar e etc.
