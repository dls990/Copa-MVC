Gerenciamento sistemático da copa

O sistema foi feito para analisar os dados dos jogos que estão ocorrendo, podem ser criados jogos quando quiser, colocando o placar, os times, seleções, e etc, utilizando um método de classificação automática, assim que criar um grupo, ele já aparece em outras paginas.

O projeto foi separado 8 pastas, sendo duas delas fora do sistema principal, apenas para mostrar o site em funcionamento, os outros 6 são o código principal do site, todas dentro da pasta "app".

O nosso banco de dados também recebe imediatamente os dados das partes que você coloca os times jogadores, contando gols, placar e etc.
